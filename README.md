# 📚 Data Structures Repository

Welcome to the **Data Structures Repository**! 🚀  
This repository contains implementations of fundamental data structures for learning, practice, and interview preparation.

---

## 🧠 Topics Covered

- Arrays
- Linked Lists (Singly, Doubly, Circular)
- Stacks
- Queues (Simple, Circular, Priority Queue)
- Trees (Binary Tree, Binary Search Tree)
- Graphs (BFS, DFS)
- Hashing
- Heaps (Min Heap, Max Heap)

---
